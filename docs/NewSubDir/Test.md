## Code Blocks

Inline code: `console.log('Hello, World!');`

```python
# Python code example
def greet(name):
    return f"Hello, {name}!"

print(greet("Alice"))
```

```javascript
// JavaScript code example
function greet(name) {
    return `Hello, ${name}!`;
}

console.log(greet("Alice"));
```
